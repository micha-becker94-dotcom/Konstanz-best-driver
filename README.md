# Grab Fahrer Konstanz – Landingpage

## So veröffentlichst du es mit GitHub Pages
1. Neues Repo erstellen (z.B. `grab-konstanz`).
2. Diese Dateien hochladen:
   - `index.html`
   - `styles.css`
   - Ordner `assets/` (Bilder)
3. Repo → **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Speichern – danach ist die Seite unter der Pages-URL erreichbar.

## Bilder ersetzen
Lege neue Bilder im Ordner `assets/` ab und behalte die Dateinamen:
- `driver-scooter.webp` / `driver-scooter.jpg`
- `driver-pointing.webp` / `driver-pointing.jpg`
